A set of internal tools developed at Avaloq to simplify integration workflows between core banking modules and external services.

## Overview

The toolkit provides reusable components for:

- **Database migration scripts** with rollback support
- **API adapters** for translating between internal and third-party data formats
- **Batch processing pipelines** for nightly reconciliation jobs

## Impact

Reduced integration setup time for new client onboarding by standardizing the patterns and eliminating repetitive boilerplate across projects.
