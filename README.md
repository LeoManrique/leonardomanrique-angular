# Leonardo Manrique - Portfolio

A modern portfolio website built with React, TypeScript, and Vite.

## Tech Stack

- React 18
- TypeScript
- Vite
- Data-driven content (JSON)

## Development

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build

# Type check
pnpm type-check
```

## Deployment

Automatically deploys to production via GitHub Actions on push to `master` branch.
