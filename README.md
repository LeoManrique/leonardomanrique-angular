# Leonardo Manrique - Portfolio

A personal portfolio website built with SvelteKit and TypeScript, featuring file-based routing and static site generation.

## Tech Stack

- SvelteKit (Svelte 5)
- TypeScript
- Vite
- adapter-static (SSG)
- Data-driven content (JSON)

## Pages

- `/` — Home, About, Qualification
- `/portfolio` — Portfolio (coming soon)
- `/blog` — Blog (coming soon)

## Development

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build

# Type check
pnpm check
```

## Deployment

Automatically deploys to production via GitHub Actions on push to `master` branch.
